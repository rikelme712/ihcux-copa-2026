# CopaNaMão UX

Integrantes
Rikelme Lourenço Gonçalves Souza

## Problema focado
Filas longas nos quiosques e perda de lances importantes durante o jogo.

## Justificativa de Design
O aplicativo foi projetado com foco em rapidez e simplicidade, utilizando botões grandes, poucas informações por tela e navegaçao direta. Isso facilita o uso em ambientes barulhentos e com distraçoes constantes, como um estádio.

## Fluxo do Usuário - Pedido de comida
1. Acessa a tela inicial (Home)
2. Clica em "Pedir comida"
3. Escolhe os itens no cardápio
4. Vai para o carrinho
5. Finaliza o pedido

## Fluxo do Usuário - Replays
1. Acessa a tela inicial (Home)
2. Clica em "Replays"
3. Seleciona o lance
4. Assiste ao replay
